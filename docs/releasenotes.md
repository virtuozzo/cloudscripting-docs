# Release Notes 0.9

### Improvements

The major improvements were applied to the [*Visual Settings*](creating-manifest/visual-settings/). New CS elements were added for a custom user-defined form.

- New elements were added to the [user input form](creating-manifest/visual-settings/):
    - [displayfield](creating-manifest/visual-settings/#displayfield) (*spacer* is an alias)
    - [spinner](creating-manifest/visual-settings/#spinner)
    - [numberpicker](creating-manifest/visual-settings/#numberpicker) (*number-picker* is an alias)
    - [hostpicker](creating-manifest/visual-settings/#hostpicker) (*host-picker* is an alias)
    - [toggle](creating-manifest/visual-settings/#toggle)
- New general configurations were added to the [user input form](creating-manifest/visual-settings/):
    - *margins*
    - *width*
    - *height*
